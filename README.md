# serene-namu-recent

나무위키 최근편집 항목들을 더 역동적으로 보여주는 웹 뷰어입니다.

## 요구되는 파이썬 모듈

* flask
* flask_restful
* requests
* beautifulsoup4
* datetime
* pytz

## API Server 실행

- <code>python server/server.py</code>
